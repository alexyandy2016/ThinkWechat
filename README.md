#ThinkWechat
==============

为thinkphp提供的微信SDK，集成opensns管理后台，wechat-php-sdk, WEUI 实现快速开发微信企业号，微信服务号，钉钉办公套件，内置二维码生成，邮件短信微信消息发送，也许是thinkphp最好最快的微信开发工具。
